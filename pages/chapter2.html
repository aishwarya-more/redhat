What is Vim?


Vim is just a simple text editor. It is similar to other text editors such as sublime text, notepad etc. Vim is also called as programmers editor. As it is a command line editor it does not have any GUI because of which it is faster and efficient to use. There are various other command line editor like nano, pico, Emacs etc. You can try and use any one of them. We will be using vim in all of our tutorials.

In order to check whether vim is installed on your machine or not type the following command in your terminal.


$ vim


If it is already installed on your machine you can see the version number. In case if it is not installed you will see an something like this vim: command not found. In order to install vim type following command and hit enter.


$ sudo dnf install vim

Note: If you are using RHEL(Red Hat Enterprise Linux) replace the above command with sudo yum install vim.

After successfully installing vim on your machine now is the time to explore it. Vim is a really powerfull editor. Once you get to know all the shortcuts/commands it will be really fast and easy to use it. Now, type the following command(without quotes) to open an existing file or create a new file.


$ vim "filename"


You will see a complete black screen if it's a new file otherwise you will be able to see the contents. Vim editor consists of different modes namely normal mode, insert mode, command mode and replace mode. To find out more visit Vim Mode.
