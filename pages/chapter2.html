What is Vim?


Vim is just a simple text editor. It is similar to other text editors such as sublime text, notepad etc. Vim is also called as programmers editor. As it is a command line editor it does not have any GUI because of which it is faster and efficient to use. There are various other command line editor like nano, pico, Emacs etc. You can try and use any one of them. We will be using vim in all of our tutorials.

In order to check whether vim is installed on your machine or not type the following command in your terminal.


$ vim --version
