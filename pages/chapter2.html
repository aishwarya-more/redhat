What is Vim?


Vim is just a simple text editor. It is similar to other text editors such as sublime text, notepad etc. Vim is also called as programmers editor. As it is a command line editor it does not have any GUI because of which it is faster and efficient to use. There are various other command line editor like nano, pico, Emacs etc. You can try and use any one of them. We will be using vim in all of our tutorials.

In order to check whether vim is installed on your machine or not type the following command in your terminal.


$ vim


If it is already installed on your machine you can see the version number. In case if it is not installed you will see an something like this vim: command not found. In order to install vim type following command and hit enter.


$ sudo dnf install vim

Note: If you are using RHEL(Red Hat Enterprise Linux) replace the above command with sudo yum install vim.

After successfully installing vim on your machine now is the time to explore it. Vim is a really powerfull editor. Once you get to know all the shortcuts/commands it will be really fast and easy to use it. Now, type the following command(without quotes) to open an existing file or create a new file.


$ vim "filename"


You will see a complete black screen if it's a new file otherwise you will be able to see the contents. Vim editor consists of different modes namely normal mode, insert mode, command mode and replace mode. To find out more visit Vim Modes.

When you open any file with vim command, initially it is in normal mode. In this mode you cannot make any changes. In order to do so you need to enter into insert mode by pressing i. In this mode you can edit the file just like any other editor.


Basic file operations in vim

Performing file related operations such as copy, paste, search, delete are little bit different in vim as compared to editors like gedit. In order to perform these operations you should enter into the command mode from insert mode by pressing Esc. We will first see how to cut, copy and paste contents in a file using vim editor.

  • Copy and Paste a line: Move the cursor at the beginning of the line which you want to copy and press yy in the command mode and then press p to paste the line which you have copied.

  • Copy and Paste multiple lines: Move the cursor at the beginning of the first line from which you want to copy and press yy+(no. of lines). For eg: yy+10 here 10 is the number of lines which you want to copy.

  • Copy and Paste a paragraph: Move the cursor at the beginning of the paragraph and press 'y+Shift+{' and press 'y+Shift+}'.

  • Cut and Paste a line/paragraph: Move the cursor at the beginning of the line which you want to cut and press cc in the command mode and then press p to paste the line.

  • Delete a line/paragraph: Move the cursor at the beginning of the line which you want to delete and press dd.

  • Delete multiple lines: Move the cursor at the beginning of the first line from which you want to delete and press Shift+v and move towards the end of the line upto which you want to delete and press d.

  • Undo: Press u to undo any operation.

Other operations in vim

In order to execute the below operations you need to enter into the command mode. Now if you are in insert mode press Esc and then press colon(:) otherwise simple press colon(:).

  • Search: To search a particular word in the file type '/keyword' and press n to find next apperance of that keyword.

  • Show the line number: To visible the line numbers in file type 'se nu'. Similarly in order to remove the line number type 'se nonu'.
