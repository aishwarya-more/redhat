Introduction


In practical scenario there can be multiple users using the environment. Linux is a multi-user operating system. In order to do so we need to create different identity for every user in working environment. In every Linux system there is one administrator user also known as superuser. Superuser is the most privileged user and has all the rights and permissions. It goes by the name of root. This name is reserved and cannot be used by any other user or group. Each and every account is unique.


Creating Users

We will be using useradd utility(command) to create an account for user using command line. In order to create a account type this utility name along with the username which you want to associate with the account. There are various other options which can be included while creating a user. For example, in the below condition a user named student with userid 1001 is created. This account expires on 31st December 2018, which is enabled with parameter -e. After this date user will be unable to login into this account.


user@Desktop:~$ useradd -e 2018-12-31 -u 1001 student


You can see the list of the users in the system using following command


user@Desktop:~$ vim /etc/passwd
